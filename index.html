1. Add the link the the JS file to your header

<script src="jquery.reflectile.js"></script>

2. Add the following javascript to the element that has a background you wish to apply this effect to:

$(document).ready(function(){	
	$('#element').reflectile({ width: '500px' });
});

Tip: Make sure to specify the width of the image

3. Make sure that the background image element is positioned either 'relative' or 'absolute':

#element{
	position:relative;
}

4. That's it, your background tile should reflect every other image giving it a nicer flow.

Note: This plugin only supports images that are positioned 'left', 'center', or 'right'.
